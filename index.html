Understanding CSS Selectors

Different Ways to Target Elements

1. The element selector applies rules to tags directly by their name (like all <p> tags).

  1. Class selectors match elements that share the same class attribute.
  2. ID selectors apply styles to a specific element with a unique id.
  3. Attribute selectors let you target elements based on attributes and their values.
  4. The universal selector * matches every element on the page.
